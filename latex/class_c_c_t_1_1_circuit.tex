\doxysubsection{Descrição detalhada}
Declaração da classe \mbox{\hyperlink{class_c_c_t_1_1_circuit}{Circuit}}. 

Representa um circuito de componentes eletrônicos através de uma matriz de incidência. Esta classe herda da classe incidence\+Matrix.
